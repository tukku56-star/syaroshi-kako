# drrrkari.com Patrol Bot

drrrkari.com の各ルームを自動巡回し、ログをJSON形式で保存するボットです。

## 必要要件
- Python 3.x
- Google Chrome
- インターネット接続

## セットアップ
必要なライブラリをインストールします：
```bash
pip install selenium webdriver-manager
```

## 実行方法
以下のコマンドでボットを起動します：
```bash
python selenium_patrol_bot.py
```

## 動作仕様
1. **ログイン**: `PatrolBot` という名前で自動ログインします。
2. **巡回**: ラウンジにある部屋を上から順に訪問します。
3. **収集**: 各部屋の参加者リストとチャットログを収集します。
4. **保存**: `logs/` ディレクトリに JSON ファイルとして保存します。
    - ファイル名: `logs/{部屋名}_{タイムスタンプ}.json`
5. **ループ**: 全部屋の巡回が終わると、再度ラウンジを更新して最初から繰り返します（無限ループ）。

## 停止方法
コンソールで `Ctrl+C` を押して停止してください。

## 注意事項
- サーバーへの負荷を考慮し、各動作の間に待機時間を設けています。
- `logs` ディレクトリは自動的に作成されます。
